# Neuromuse
Neuromuse is a web application that generates customizable frequency tones and ambient soundscapes using the Web Audio API.

## Features
- Real-time frequency tone generation

## Tech Stack
- React.js
- Web Audio API
- JavaScript

## Live Demo
https://neuromusee.netlify.app/

## Installation
git clone https://github.com/Misbah374/neuromuse.git<br>
cd neuromuse<br>
npm install<br>
npm start<br>
